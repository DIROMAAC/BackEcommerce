# proyecto-server

Backend de GymStyle construido con Node.js, Express y MongoDB.

## Requisitos

- Node.js
- MongoDB local en `mongodb://localhost:27017/proyecto_db`

## Instalacion

```bash
npm install
```

## Ejecucion

```bash
npm run dev
```

## Variables de entorno

Crear un archivo `.env` en la raiz del proyecto con las variables que use el backend, por ejemplo:

```env
PORT=8080
SECRET_KEY=tu_clave_secreta
```
